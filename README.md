# OpenLibrary

## Requirements:
* iOS 13+ support
* List of books (title, cover image, first publication date)
* Book details (title, description, first publication date, cover image, average user rating)

## Features:
* Adoptive UI
* Multithreaded data processing
* VIPER architecture
* User friendly interface

## Technology stack:
* UIKit
* URLSession
* DispatchQueue
